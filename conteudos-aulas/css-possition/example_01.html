CSS Positioning Examples

Static

Static is the default positioning. The element stays in the normal document flow.

Relative

Relative positioning moves the element relative to its original position.

Fixed

Fixed positioning keeps the element in the same spot, even when scrolling.

Absolute

Absolute positioning moves the element relative to the nearest positioned ancestor.

Sticky

Scroll down to see the sticky element in action. It behaves like relative until it reaches the defined offset, then it sticks to the top of the scroll container.
